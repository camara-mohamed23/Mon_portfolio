Détail de mon projet plateforme de vente en ligne avec Spring Boot

  1. Accueil
  2. Détail du projet

Travail effectué

  • Projet : Développement d’une plateforme complète de vente en ligne avec Spring Boot (backend) et Angular standalone (frontend)
  • Fonctionnalités principales :
    • Catalogue produits avec catégories, stock, images et gestion CRUD (admin)
    • Panier d’achat interactif (ajout, mise à jour, suppression d’articles)
    • Commandes & paiement (simulation) avec suivi des statuts de transactions
    • Gestion des utilisateurs (inscription, connexion, rôles admin/client) sécurisée par JWT
    • Notifications email automatiques (confirmation de commande)
    • Frontend moderne Angular (HTML, CSS, TypeScript) connecté à l’API REST
    • Base de données MySQL avec Spring Data JPA & Hibernate

Impact du projet

  • Impact : Fourniture d’une solution e-commerce complète et sécurisée, démontrant la maîtrise du développement full-stack moderne, de l’architecture modulaire et des bonnes pratiques de sécurité pour des applications web professionnelles.

    • Technologies utilisées

  • Frontend moderne Angular (HTML, CSS, TypeScript) connecté à l’API REST
  • Base de données MySQL avec Spring Data JPA & Hibernate

    • Compétences démontrées :

    • Développement full-stack (Java Spring Boot + Angular)
    • Mise en place de la sécurité avec Spring Security + JWT
    • Conception d’une API RESTful et intégration avec un frontend moderne
    • Gestion d’une architecture modulaire (produits, panier, commandes, paiements)
    • Intégration de MySQL et configuration via application.properties
    • Déploiement local avec Tomcat intégré et frontend Angular standalone
